CSS3 property animation

animation: infinite myMove 3s alternate;
animation-name: myMove; // animationName
animation-iteration-count: infinite;
// count of iterations
animation-duration: 3s;
// duration

CSS3 property animation

animation-direction: alternate;
// direction
animation-delay: 5s; // delay
animation-timing-function: ease;
// acceleration function
animation-timing-function examples

The rule @keyframes

0% - Element styles before animation
100% - Element styles in the end of animation 

         @keyframes animationName {
            0%   { background: red; }
            50%  { background: blue; }
            100% { background: green; } }
