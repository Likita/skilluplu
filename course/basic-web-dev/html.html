Semantic markup

The approach of using tags according to their purpose

<em>, <strong> - semantic selection of text
<i>, <b> - just appearance

Semantic markup. Advantages

  • The code is better analyzed by search engines
  • Good positions in search engine results
  • Fewer items - faster to download
  • Correct recognition by voice browsers
