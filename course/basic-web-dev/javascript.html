Variable

We need variables to store data

We can change the value of variables

Variable declaration

let count = 5 - any variable
const count = 5 - only constant
var count = 5 - old syntax for any variable

count - the name of the variable

= - an assignment operator

5 - the value of the variable

Declaring a string variable

name = "Vasilika" - double quotes
name = 'Vasilika' - single quotes
name = `Vasilika` - template literals are enclosed by the back-tick ` (grave accent)

name - the name of the variable

= - an assignment operator

"Vasilika" - the value of the variable

How to find name for a variable?

  • The first character must not be a number
  • May consist of latin letters, numbers, symbols $ and _
  • Case of letters matters. day ≠ Day
  • There are reserved words
  • The name should reflect the meaning of the content
  • Clear to everyone without any comments
  • As short as possible
