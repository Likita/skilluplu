Переменные нужны, чтобы хранить данные

Значение переменных можно менять

Объявление переменной

k = 5

k - это название переменной

= это оператор присваивания

5 - это значение переменной

Объявление переменной

message = "Привет"

message - это название переменной

= это оператор присваивания

"Привет" - это значение переменной

Именование переменной

  • Первый символ не должен быть цифрой
  • Может состоять из букв, цифр, символов $ и _
  • Регистр букв имеет значение
  • Есть зарезервированные слова
day ≠ Day
